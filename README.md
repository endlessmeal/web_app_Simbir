# Web App with 5 routes:
```
/animal/cat
/animal/dog
/animal/fox
/history
/history/static/<uuid>
```
### Installing
What things you need to install the software and how to install them

```
pip3 install Flask
pip3 install flask-sqlalchemy
```

### Launching
Run it in bash with cd /your/path/to/project

```
python3 views.py
```
